Older Versions

It took me quite a while to collect all of these postcards...

How was it created?

  1. All postcards were mass scanned using a script I wrote that could scan 4 postcards at a time (one in each corner of my scanner) while I did other work. I wrote the script for scanning sets of photographs, and making it work for postcards was trivial. This is the reason that many of the postcard images aren't perfectly cropped though.
  2. A single postcard was repeatedly used to lay out the diagram using xfig.
  3. The xfig save file (which is ascii) was converted to reference all the postcards rather than just the single repeated one, and then used to turn the layout into a jpeg.
  4. The drop shadow was added using gimp.
  5. The image map was created from a perl script that translated the xfig postcard coordinates into an html image map enabling people to click on each postcards/postcard.

A side affect to this process is that xfig outputs its save file in fairly random order, so everytime I create a new logo the postcards are randomly placed. The only ones that are consistent are the 2 composing the dash in the center, which are a postcard from UCDavis and a picture of the first ucd-snmp roundtable.
