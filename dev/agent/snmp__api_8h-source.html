00627     /*

00634      * Returns info about what snmp requires from a select statement.

00636      * All file descriptors opened for SNMP are OR'd into the fdset.
00637      * If activity occurs on any of these file descriptors, snmp_read

00639      *
00640      * The timeout is the latest time that SNMP can wait for a timeout.  The

00646      * Block is 1 if the select is requested to block indefinitely, rather than
00647      * time out.  If block is input as 1, the timeout value will be treated as
00648      * undefined, but it must be available for setting in snmp_select_info.  On
00649      * return, if block is true, the value of timeout will be undefined.
00650      *
00651      * snmp_select_info returns the number of open sockets.  (i.e. The number
