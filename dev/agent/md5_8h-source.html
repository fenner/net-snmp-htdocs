00034      * MDupdate(MD,X,count)
00035      * ** Input: MD -- an MDptr
00036      * **        X -- a pointer to an array of unsigned characters.
00037      * **        count -- the number of bits of X to use (an unsigned int).
00038      * ** Updates MD using the first ``count'' bits of X.

00040      * ** If count is not a multiple of 8, MDupdate uses high bits of last byte.
