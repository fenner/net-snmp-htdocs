GIT

The Net-SNMP project switched from SVN to GIT in June 2011. We then moved to using the new 'allura' SourceForge framework in December 2012, which also affected the URLs used to retrieve the code.
Please ensure that you use the repository shown below - using anything else may result in obtaining an old version of the project code.

The source GIT tree can be browsed via the Sourceforge GIT repository. This also gives information about retrieving a local copy of the source code, or see the Net-SNMP Git wiki page.
